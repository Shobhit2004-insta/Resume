# 📄 Resume Website

This is a single-page resume website built using HTML and CSS to present my personal details, skills, education, and projects in a clean and professional format.

---

## 🚀 Features

- 📄 One-page resume layout  
- 👤 Profile and contact information  
- 🛠 Technical skills section  
- 🎓 Education details  
- 💻 Projects with Live Demo & GitHub links  
- 📱 Clean and print-friendly design  

---

## 🛠 Technologies Used

- HTML  
- CSS  

---

## 📂 Project Structure

Resume-Website
│
├── index.html
└── style.css

---

## 💡 Purpose

This project is designed to showcase my resume in a web format, making it easy to share online and view across devices.

---

## 🎯 Learning Outcome

- Structuring content using HTML  
- Designing layouts with CSS  
- Creating a clean and readable UI  
- Making a print-friendly (PDF) resume
